# Agnes AI 提示词管理工具

版本 1.2.0 | 2026-06-23
项目位置: `agnes-ai-client/`

## 一、项目概述

集成 AI 图像/视频生成、提示词管理和专业提示词增强的一站式平台。接入 Agnes AI API（Image 2.1 Flash、Video V2.0、2.0 Flash），在统一界面中完成从自然语言输入到专业提示词生成、图像/视频生成、提示词分类存储的完整工作流。

## 二、技术架构

前端: HTML5 + CSS3 + JavaScript（纯前端，无框架依赖）
前端服务器: Python Flask（静态文件服务 + API代理）
数据库: SQLite（提示词持久化存储）
AI API: Agnes AI（图像生成、视频生成、文本对话、提示词增强）
加密: Fernet（API Key安全存储）
部署: 本机运行（普通笔记本电脑/台式机）或打包为独立exe

## 四、数据库设计

当前结构:
```sql
CREATE TABLE prompts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    text TEXT, title_cn TEXT, category TEXT,
    tags TEXT, style TEXT, usage_ TEXT,
    favorite INTEGER DEFAULT 0, source_tab TEXT,
    created_at TEXT, updated_at TEXT
);
```

计划扩展字段:
- 摄影参数: focal_length, aperture, lighting, composition, perspective, color_tone, quality_style
- 视频参数: shot_type, camera_movement, transition, focus_technique, character_action, character_expression, mood_atmosphere
- 交互: rating（评分1-5）, notes（备注）

## 五、API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET/POST | /api/config | API Key 配置 |
| POST | /api/v1/images/generations | 文生图/图生图代理 |
| POST | /api/v1/videos | 文生视频/图生视频代理 |
| GET | /api/videos/query | 视频结果查询 |
| POST | /api/v1/chat/completions | 文本对话代理 |
| GET/POST | /api/prompts | 提示词列表/保存 |
| GET/PUT/DELETE | /api/prompts/:id | 单条提示词操作 |
| POST | /api/prompts/analyze | 提示词分析 |
| POST | /api/prompts/enhance | 提示词增强 |
| POST | /api/prompts/reverse | 图片反推提示词 |
| POST | /save-image | 图片缓存 |
| GET | /generated-images/:file | 图片服务 |

## 七、部署说明

环境: Python 3.8+, 内存 ≥4GB, 需访问 https://apihub.agnes-ai.com

安装依赖:
```bash
pip install -r requirements.txt
```

启动:
```bash
python app/server.py
```

访问: http://localhost:3001

首次使用会提示输入 API Key，保存在后端 config.json。

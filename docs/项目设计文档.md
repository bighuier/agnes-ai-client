# Agnes AI 项目设计文档

## 1. 项目概述

### 1.1 项目简介

Agnes AI 是一个基于 Flask 框架开发的 AI 图像/视频生成 Web 应用，提供文生图、图生图、文生视频、图生视频等多种 AI 生成能力，并支持提示词管理功能。

### 1.2 核心功能

| 功能模块 | 描述 |
|---------|------|
| 文生图 | 通过文字描述生成图片 |
| 图生图 | 基于上传图片进行图像编辑 |
| 文生视频 | 通过文字描述生成视频 |
| 图生视频 | 基于上传图片生成视频 |
| 提示词库 | 管理和复用 AI 提示词 |

### 1.3 技术栈

| 分类 | 技术 | 版本 |
|------|------|------|
| 后端框架 | Flask | 2.x |
| 数据库 | SQLite | 3.x |
| 前端 | HTML5 + JavaScript | ES6+ |
| 外部 API | Agnes AI API | - |

### 2.2 核心流程图

#### 图片生成流程
```
用户输入提示词 → 点击生成 → 前端调用 /api/v1/images/generations 
                    → 后端代理请求 → Agnes AI API 
                    → 返回图片URL → 前端缓存图片 → 展示结果
```

#### 提示词保存流程
```
用户点击保存 → 前端调用 /api/prompts/analyze → 分析提示词分类
                    → 调用 /api/prompts → 保存到数据库
```

## 5. API 接口文档

### 5.1 配置管理

#### GET /api/config
**描述**：获取配置状态

**响应**：
```json
{
  "has_key": true
}
```

#### POST /api/config
**描述**：保存 API Key

**请求体**：
```json
{
  "api_key": "sk-xxx"
}
```

### 5.2 图片生成

#### POST /api/v1/images/generations
**描述**：生成图片

**请求体**：
```json
{
  "model": "agnes-image-2.1-flash",
  "prompt": "一只可爱的猫咪",
  "size": "1024x768",
  "extra_body": {
    "response_format": "url"
  }
}
```

**响应**：
```json
{
  "data": [
    {
      "url": "https://example.com/image.png"
    }
  ]
}
```

### 5.3 视频生成

#### POST /api/v1/videos
**描述**：生成视频

**请求体**：
```json
{
  "model": "agnes-video-v2.0",
  "prompt": "一只奔跑的小狗",
  "width": 1152,
  "height": 768,
  "num_frames": 121,
  "frame_rate": 24
}
```

**响应**：
```json
{
  "video_id": "abc123"
}
```

#### GET /api/videos/query
**描述**：查询视频状态

**参数**：
- `video_id`: 视频ID
- `model_name`: 模型名

**响应**：
```json
{
  "status": "completed",
  "progress": 100,
  "video_url": "https://example.com/video.mp4"
}
```

### 5.4 提示词管理

#### POST /api/prompts/enhance
**描述**：增强提示词

**请求体**：
```json
{
  "text": "一只猫"
}
```

**响应**：
```json
{
  "enhanced": "一只可爱的橘猫，坐在窗台上，阳光明媚，高清照片"
}
```

#### POST /api/prompts/analyze
**描述**：分析提示词（AI）

**请求体**：
```json
{
  "text": "一只可爱的猫咪"
}
```

**响应**：
```json
{
  "title_cn": "可爱猫咪",
  "category": "动物",
  "tags": ["宠物", "猫咪"],
  "style": "写实",
  ...
}
```

#### GET /api/prompts
**描述**：获取提示词列表

**参数**：
- `q`: 搜索关键词
- `category`: 分类筛选
- `fav`: 是否收藏

**响应**：
```json
{
  "items": [...],
  "total": 10,
  "categories": {"动物": 5, "风景": 3}
}
```

#### POST /api/prompts
**描述**：创建提示词

**请求体**：
```json
{
  "text": "提示词内容",
  "title_cn": "标题",
  "category": "分类",
  "tags": ["标签1", "标签2"]
}
```

#### GET /api/prompts/{id}
**描述**：获取单个提示词

#### PUT /api/prompts/{id}
**描述**：更新提示词

**请求体**：
```json
{
  "rating": 5,
  "favorite": 1
}
```

#### DELETE /api/prompts/{id}
**描述**：删除提示词

### 5.5 分类管理

**响应**：

## 7. 安全性说明

| 风险点 | 说明 | 当前处理 |
|--------|------|----------|
| API Key 暴露 | 密钥存储在服务端 | 仅在服务端使用，不暴露给前端 |
| SQL 注入 | 数据库查询安全 | 使用参数化查询 |
| 文件上传 | 图片上传风险 | 仅允许图片类型，本地存储 |
| XSS 攻击 | 前端安全 | 使用 `esc()` 函数转义输出 |
| CORS | 跨域安全 | 允许所有来源（开发环境） |

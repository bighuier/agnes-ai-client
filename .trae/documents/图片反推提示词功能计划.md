# 图片反推提示词功能实施计划

## 概述
为Agnes AI Client添加图片反推提示词功能，用户上传图片后可以一键获取AI生成的提示词描述，用于图生图或图生视频场景。

## 当前状态分析

### 已有架构
1. **后端 ([server.py](file:///d:/测试文件夹/AI代码测试/agnes-ai-frontend/app/server.py))**
   - Flask框架，已有proxy函数处理API请求
   - 已有 `/api/prompts/enhance` 路由用于增强提示词
   - 已有 `/api/v1/chat/completions` 代理路由
   - API Key加密存储机制已完善

2. **前端 ([index.html](file:///d:/测试文件夹/AI代码测试/agnes-ai-frontend/templates/index.html))**
   - 图生图(Tab 2)和图生视频(Tab 4)页面已有图片上传功能
   - `up` 数组存储上传图片的DataURL (base64格式)
   - 已有状态提示函数 `st()`

3. **API能力**
   - Agnes AI的agnes-2.0-flash模型支持图片理解(Vision)
   - 兼容OpenAI Chat Completions协议
   - 支持图片URL和base64格式输入

### 技术方案
使用OpenAI Vision API格式：
```json
{
  "model": "agnes-2.0-flash",
  "messages": [
    {
      "role": "user",
      "content": [
        {"type": "text", "text": "分析图片并生成提示词"},
        {"type": "image_url", "image_url": {"url": "data:image/jpeg;base64,..."}}
      ]
    }
  ]
}
```

## 实施步骤

### 步骤1：后端API路由
**文件**: [server.py](file:///d:/测试文件夹/AI代码测试/agnes-ai-frontend/app/server.py)

添加新路由 `/api/prompts/reverse`：
```python
@app.route("/api/prompts/reverse", methods=["POST"])
def reverse_prompt():
    image_data = request.json.get("image", "")
    if not image_data:
        return jsonify({"error": "no image"}), 400

    # 构建Vision API请求
    sys_msg = "你是专业AI提示词工程师。分析图片并生成详细的专业提示词，包含主体描述、风格、光线、构图、色调等要素。输出以中文为主，仅保留必要的英文专业术语。"
    user_msg = "请分析这张图片，生成一个可以重现类似效果的专业AI提示词："

    r, c = proxy(AGNES + "/v1/chat/completions", {
        "model": "agnes-2.0-flash",
        "messages": [
            {"role": "system", "content": sys_msg},
            {"role": "user", "content": [
                {"type": "text", "text": user_msg},
                {"type": "image_url", "image_url": {"url": image_data}}
            ]}
        ]
    }, "POST")

    if isinstance(r, dict) and r.get("error"):
        return jsonify({"error": r.get("error")}), 400

    try:
        prompt = r["choices"][0]["message"]["content"].strip()
        return jsonify({"prompt": prompt})
    except:
        return jsonify({"error": "解析失败"}), 500
```

**位置**: 在 `@app.route("/api/prompts/enhance")` 之后添加

#### 2.3 添加JavaScript函数
在script标签内添加：
```javascript
async function reversePrompt(t) {
    // 检查是否有上传图片
    if (!up[t] || !up[t].length) {
        st("s" + t, "请先上传图片", "error");
        return;
    }

    var btn = document.querySelector("#t" + (t === 1 ? 2 : 4) + " .en-btn[onclick*='reversePrompt']");
    if (btn && btn.disabled) return;
    if (btn) btn.disabled = true;

    st("s" + t, "分析图片中...", "pending");

    try {
        var imageData = up[t][0]; // 取第一张图片的DataURL
        var r = await ap("/api/prompts/reverse", {image: imageData});

        if (r.error) {
            st("s" + t, "反推失败: " + r.error, "error");
        } else {
            var el = document.getElementById("p" + t);
            if (el) el.value = r.prompt;
            st("s" + t, "反推完成", "success");
        }
    } catch (e) {
        st("s" + t, "反推失败: " + e.message, "error");
    }

    if (btn) btn.disabled = false;
}
```

## 假设与决策

1. **假设**: Agnes AI的agnes-2.0-flash模型完全兼容OpenAI Vision API格式
2. **决策**: 只支持单张图片反推（取up数组第一张），多张图片场景较少
3. **决策**: 反推结果直接填充到提示词输入框，用户可手动修改后再使用
4. **决策**: 不自动触发图生图/图生视频，让用户确认提示词后再操作

## 风险评估

- **低风险**: API格式兼容性问题 - Agnes AI官方声明兼容OpenAI协议
- **低风险**: 图片大小限制 - DataURL可能较大，但agnes-2.0-flash支持4K图片
- **低风险**: 响应时间 - Vision请求可能较慢，已有pending状态提示
